# Java基礎知識：メソッド（Method）のまとめ

## 1. メソッドとは？

**メソッド（Method）**とは、特定の処理をひとつにまとめたものです。

同じ処理を何度も記述する必要がなくなり、コードの再利用性や保守性を高めることができます。

```java
public class Main {

    public static void main(String[] args) {
        hello();
    }

    public static void hello() {
        System.out.println("Hello");
    }
}
```

この例では、`hello()` がメソッドです。

---

## 2. メソッドの基本構造

```java
アクセス修飾子 戻り値の型 メソッド名(パラメータ) {
    処理
}
```

例：

```java
public static void hello() {
    System.out.println("Hello");
}
```

---

## 3. メソッドを構成する要素

### アクセス修飾子（Access Modifier）

メソッドへアクセスできる範囲を指定します。

主な種類：

* `public`
* `protected`
* （デフォルト）
* `private`

例：

```java
public void hello() {
}
```

---

### 戻り値の型（Return Type）

メソッドが呼び出し元へ返す値の型を指定します。

#### void

戻り値がない場合に使用します。

```java
public void hello() {
    System.out.println("Hello");
}
```

このメソッドは値を返しません。

---

### メソッド名（Method Name）

メソッドを識別するための名前です。

```java
public void hello() {
}
```

この場合のメソッド名は `hello` です。

---

### パラメータ（Parameter）

パラメータとは、メソッド定義時に宣言する変数です。

メソッド呼び出し時に渡された引数を受け取るために使用します。

```java
public void printName(String name) {
    System.out.println(name);
}
```

この場合の `name` がパラメータです。

---

## 4. mainメソッドとは？

Javaプログラムの実行開始時に最初に呼び出される特別なメソッドです。

```java
public static void main(String[] args) {
}
```

JVMはこのメソッドを実行することでプログラムを開始します。

> **mainメソッドはJavaプログラムのエントリーポイントです。**

---

## 5. public static void main(String[] args) の意味

### public

JVMから呼び出せるようにするために使用します。

```java
public static void main(String[] args)
```

---

### static

オブジェクトを生成しなくても実行できるようにするために使用します。

プログラム開始時にはまだオブジェクトが存在しないため、mainメソッドにはstaticが必要です。

```java
public static void main(String[] args)
```

---

### void

戻り値を返さないことを表します。

```java
public static void main(String[] args)
```

mainメソッドは処理結果を呼び出し元へ返さないため、`void` を使用します。

---

### main

Javaプログラムの開始地点となる特別なメソッド名です。

```java
public static void main(String[] args)
```

JVMは `main` という名前のメソッドを探して実行します。

---

### String[] args

コマンドライン引数を受け取るためのパラメータです。

```java
public static void main(String[] args)
```

プログラム実行時に渡されたコマンドライン引数が配列として格納されます。

例えば、

```bash
java Main Tokyo Osaka
```

と実行すると、

```java
args[0] → "Tokyo"
args[1] → "Osaka"
```

が格納されます。

---

## 6. 引数（Argument）・パラメータ（Parameter）・戻り値（Return Value）

### パラメータ（Parameter）

パラメータとは、メソッド定義時に宣言する変数のことです。

```java
public static void printName(String name) {
    System.out.println(name);
}
```

この場合、

```java
String name
```

の `name` がパラメータです。

---

### 引数（Argument）

引数とは、メソッド呼び出し時に渡す値のことです。

```java
printName("Tanaka");
```

この場合、

```java
"Tanaka"
```

が引数です。

---

### 引数とパラメータの違い

Javaでは「引数」と「パラメータ」が同じ意味で使われることもありますが、厳密には意味が異なります。

```java
public static void add(int a, int b) {
    System.out.println(a + b);
}

add(10, 20);
```

| 種類    | 内容               |
| :---- | :--------------- |
| パラメータ | `int a`, `int b` |
| 引数    | `10`, `20`       |

> 実務では両者をまとめて「引数」と呼ぶこともありますが、厳密には意味が異なります。

---

### 戻り値（Return Value）

戻り値とは、メソッドが処理結果を呼び出し元へ返す値のことです。

```java
public static int add(int a, int b) {
    return a + b;
}
```

この場合、

```java
return a + b;
```

によって計算結果が呼び出し元へ返されます。

---

### 引数あり・戻り値なし

```java
public static void printName(String name) {
    System.out.println(name);
}
```

---

### 引数なし・戻り値あり

```java
public static int getNumber() {
    return 100;
}
```

---

### 引数あり・戻り値あり

```java
public static int add(int a, int b) {
    return a + b;
}
```

## 8. 面接でよく深掘りされる追加質問（実務知識）

### Q. オーバーロードとは何ですか？

**A.**
同じメソッド名で引数の型や数を変えて複数のメソッドを定義することです。

```java
public int add(int a, int b) {
    return a + b;
}

public double add(double a, double b) {
    return a + b;
}
```

* **面接での補足ポイント:**
  戻り値だけを変更したオーバーロードはできません。

### Q. return文の役割は何ですか？

**A.**
メソッドの処理結果を呼び出し元へ返すために使用します。

```java
public int getNumber() {
    return 100;
}
```

* **面接での補足ポイント:**
  `return` が実行された時点でメソッドの処理は終了します。

### Q. staticメソッドとインスタンスメソッドの違いは何ですか？

**A.**
staticメソッドはオブジェクト生成なしで利用でき、インスタンスメソッドはオブジェクト生成後に利用します。

```java
class Sample {

    static void methodA() {
    }

    void methodB() {
    }
}
```

* **面接での補足ポイント:**
  staticメソッドからインスタンス変数やインスタンスメソッドへ直接アクセスすることはできません。

### Q. なぜメソッドを使うのですか？

**A.**
処理をひとつにまとめることで、コードの再利用性や保守性を向上させるためです。

* **面接での補足ポイント:**
  同じ処理を複数回記述する必要がなくなり、修正箇所を減らすことができます。

---

## 9. 面接用まとめ

* メソッド：処理をまとめたもの
* パラメータ：メソッド定義時に宣言する変数
* 引数：メソッド呼び出し時に渡す値
* 戻り値：メソッドが返す値
* void：戻り値なし
* main：Javaプログラムの開始地点
* String[] args：コマンドライン引数を受け取るパラメータ
* static：オブジェクト生成なしで利用可能

一言で言うと、

> **メソッドとは、特定の処理をまとめて再利用できるようにしたものであり、Javaプログラムはmainメソッドから実行が開始されます。**
