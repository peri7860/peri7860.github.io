# Java基礎知識：Mathクラス（乱数・四捨五入）のまとめ

## 1. Math.random() とは？

`Math.random()` は、

```java
0.0以上 1.0未満
```

のランダムな `double` 型の値を返すメソッドです。

```java
System.out.println(Math.random());
```

実行するたびに異なる値が出力されます。

---

## 2. 乱数の利用例

乱数は次のような場面でよく使用されます。

* ロト・くじ番号の生成
* じゃんけんゲーム
* サイコロゲーム
* アップダウンゲーム
* 認証コードの生成

---

## 3. サイコロの目（1～6）の生成

サイコロは 1～6 の数字をランダムに生成する必要があります。

### 基本式

```java
(int)(Math.random() * 6) + 1
```

### 処理の流れ

① `Math.random()` を使う

```java
0.0 <= Math.random() < 1.0
```

② 6を掛ける

```java
0.0 <= Math.random() * 6 < 6.0
```

③ int型へ変換する

```java
0 ～ 5
```

④ 1を足す

```java
1 ～ 6
```

### 使用例

```java
int dice = (int)(Math.random() * 6) + 1;

System.out.println(dice);
```

---

## 4. 指定範囲の整数を生成する公式

### 1～N

```java
(int)(Math.random() * N) + 1
```

例：

```java
1 ～ 10
```

```java
int num = (int)(Math.random() * 10) + 1;
```

---

### min～max

```java
(int)(Math.random() * (max - min + 1)) + min
```

例：

```java
10 ～ 20
```

```java
int num =
    (int)(Math.random() * 11) + 10;
```

---

## 5. じゃんけんゲームの例

以下のように数値を割り当てます。

| 数値 | 手 |
| :--- | :--- |
| 1 | グー |
| 2 | チョキ |
| 3 | パー |

### コンピュータの手を生成

```java
int com =
    (int)(Math.random() * 3) + 1;
```

結果：

```java
1 ～ 3
```

のいずれかが生成されます。

### 勝敗判定の例

```java
int win = me - com;
```

プレイヤーとコンピュータの差を利用して勝敗を判定できます。

---

### アルゴリズムの補足知識

じゃんけんの勝敗判定では、差分を利用する方法以外に、剰余（%）演算を利用する方法もあります。

```java
int result = (me - com + 3) % 3;
```

結果は常に以下のいずれかになります。

| result | 判定 |
| :--- | :--- |
| 0 | あいこ |
| 1 | 負け |
| 2 | 勝ち |

例：

```java
int result = (me - com + 3) % 3;

if(result == 0){
    System.out.println("あいこ");
}else if(result == 1){
    System.out.println("負け");
}else{
    System.out.println("勝ち");
}
```

この方法は条件分岐をシンプルにできるため、アルゴリズム問題などでよく利用されます。

## 6. Math.round() とは？

`Math.round()` は四捨五入を行うメソッドです。

```java
Math.round(3.6)
```

結果：

```text
4
```

---

## 7. 小数点以下を四捨五入する方法

### 小数第1位まで表示

```java
double num = 3.141592;

double result =
    Math.round(num * 10) / 10.0;
```

結果：

```text
3.1
```

---

### 処理の流れ

① 10を掛ける

```java
3.141592 × 10
↓
31.41592
```

② 四捨五入

```java
Math.round(31.41592)
↓
31
```

③ 10.0で割る

```java
31 / 10.0
↓
3.1
```

---

### 小数第2位まで表示

```java
double result =
    Math.round(num * 100) / 100.0;
```

結果：

```text
3.14
```

## 乱数生成に関する補足

`Math.random()` は学習用プログラムやゲームなどでは十分に利用できます。

しかし、

* 認証コード
* ワンタイムパスワード（OTP）
* パスワード生成
* セッションID

などのセキュリティが重要な用途では使用しません。

実務では、

```java
java.security.SecureRandom
```

を使用するのが一般的です。

```java
import java.security.SecureRandom;

SecureRandom random =
    new SecureRandom();

int code =
    random.nextInt(1000000);
```

`SecureRandom` は予測されにくい乱数を生成できるため、セキュリティ用途に適しています。

### 面接での補足ポイント

> 「乱数生成には Math.random() が使用できますが、認証コードやパスワードなどのセキュリティ用途では SecureRandom を使用するのが一般的です。」

と説明できると、実務を意識した知識として評価されることがあります。

## 9. まとめ

* `Math.random()` は乱数を生成する
* 戻り値は 0.0以上1.0未満の double型
* サイコロは `(int)(Math.random() * 6) + 1`
* 指定範囲の整数も生成できる
* `Math.round()` は四捨五入を行う
* 小数点以下を丸める場合は「掛ける → round → 割る」

一言で言うと、

> **Math.random() は乱数生成、Math.round() は四捨五入に使用するメソッドであり、Javaの基礎学習で頻繁に利用されます。**
