# String補足（文字列の比較と参照）

## 1. Stringは参照型（Reference Type）

`String` は文字列を扱うためのクラスであり、参照型です。

変数には文字列そのものではなく、文字列オブジェクトの参照（アドレス）が格納されます。

```java
String str1 = "java";
String str2 = "java";
```

この場合、Javaは同じ文字列リテラルを再利用するため、`str1` と `str2` は同じ文字列オブジェクトを参照します。

```java
System.out.println(str1 == str2);
```

結果：

```text
true
```

---

## 2. == は何を比較しているのか？

`==` は文字列の内容ではなく、参照先（アドレス）が同じかどうかを比較します。

```java
String str1 = "java";
String str2 = "java";

System.out.println(str1 == str2);
```

結果：

```text
true
```

これは、

```text
"java" と "java" の文字列内容が同じだから
```

ではなく、

```text
同じオブジェクトを参照しているから
```

です。

そのため、

> `==` は文字列の内容を比較する演算子ではなく、参照先が同じかどうかを比較する演算子です。

---

## 3. new String() を使用した場合

Stringは本来クラスであるため、`new` を使用してオブジェクトを生成できます。

```java
String str1 = "java";
String str3 = new String("java");
```

この場合、

```java
System.out.println(str1 == str3);
```

結果：

```text
false
```

となります。

---

### なぜ false になるのか？

`new String("java")` を実行すると、新しいStringオブジェクトが生成されます。

そのため、

```text
str1 → 既存の "java" オブジェクト
str3 → 新しく生成された "java" オブジェクト
```

となり、文字列の内容は同じでも参照先が異なります。

イメージ：

```text
str1 ──→ [java] (100番地)

str3 ──→ [java] (200番地)
```

内容は同じですが、別のオブジェクトです。

---

## 4. 文字列の内容を比較する方法

文字列の内容を比較する場合は `equals()` を使用します。

```java
String str1 = "java";
String str3 = new String("java");

System.out.println(str1.equals(str3));
```

結果：

```text
true
```

`equals()` は文字列の内容そのものを比較します。

---

## 5. 実務での基本ルール

文字列比較では基本的に `equals()` を使用します。

```java
if(id.equals("admin")){
}
```

または

```java
if("admin".equals(id)){
}
```

---

### 推奨される書き方

```java
if("admin".equals(id)){
}
```

この書き方であれば、

```java
id == null
```

の場合でも `NullPointerException` を防ぐことができます。

---

## 6. まとめ

| 比較方法 | 比較対象 |
| :--- | :--- |
| `==` | 参照先（アドレス） |
| `equals()` | 文字列の内容 |

例：

```java
String a = "java";
String b = "java";
String c = new String("java");

System.out.println(a == b);       // true
System.out.println(a == c);       // false
System.out.println(a.equals(c));  // true
```

---

### ポイント

* Stringは参照型である
* `==` は参照先を比較する
* `equals()` は文字列の内容を比較する
* `new String()` を使用すると新しいオブジェクトが生成される
* 実務では文字列比較に `equals()` を使用する

> **Stringを比較するときは、参照先を比較する `==` ではなく、文字列の内容を比較する `equals()` を使用するのが基本です。**
